Kaiju Mapper
============

A python library implementing TDA Mapper pipelines seen in the wild.

.. grid:: 3

  .. grid-item-card::

     :octicon:`apps;1em;sd-text-info` Modular
     ^^^
     Pipelines are implemented as reusable composable components. This allows
     you to mix and match as desired.

  .. grid-item-card::

     :octicon:`plus-circle;1em;sd-text-info` Extensible
     ^^^
     Kaiju Mapper is implemented as a layer on top of Zen Mapper. This allows
     for pain free implementation of new techniques.

  .. grid-item-card::

     :octicon:`light-bulb;1em;sd-text-info` Examples
     ^^^
     See the :ref:`example gallery <pipeline-examples>` for examples of pipelines run with Kaiju.
